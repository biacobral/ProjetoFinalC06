# 🧙 Padrinhos Mágicos — Banco de Dados Relacional

Projeto final da disciplina de Banco de Dados (C06) — Inatel, 2025.

Modelagem e implementação de um banco de dados relacional temático
baseado no universo de Padrinhos Mágicos, com foco em estruturação
de esquemas, integridade referencial e consultas analíticas.

## Tecnologias

- MySQL
- SQL (DDL, DML, DQL)

## O que foi implementado

- Modelagem relacional com 7 tabelas e relacionamentos N:N, 1:N
- Chaves estrangeiras com ON DELETE CASCADE e ON DELETE SET NULL
- Trigger para validação de regras de negócio
- View agregada com LEFT JOIN e GROUP BY
- Consultas com múltiplos JOINs entre entidades

## Como executar

1. Abra o MySQL Workbench ou qualquer cliente MySQL
2. Execute o arquivo `projetinho.sql`
3. O banco `Padrinhos_Magicos` será criado automaticamente
